# EEOS Project Principles

## Mission

Build the best configurable Education Enterprise Operating System (EEOS) for coaching institutes.

## Current Product

EEOS Lite (Beta)

## Beta Customer

Veda EdTech

## Development Philosophy

- Configuration before customization.
- No hardcoded business rules.
- Every workflow should be configurable.
- Everything should be multi-company ready.
- Everything should be multi-branch ready.
- Everything should be role-based.
- Build once, reuse everywhere.

## Golden Rules

1. Documentation before implementation.
2. Never duplicate business logic.
3. Every architectural decision must be recorded.
4. Build reusable components first.
5. Every new module must integrate with Tasks, Notifications and Access Control where applicable.
